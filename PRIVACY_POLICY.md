# WhiteRocket Privacy Policy

Effective date: July 5, 2026

WhiteRocket is an Android VPN/proxy client that lets users import and use their own server configurations. This policy explains what data the app handles and how it is used.

## Information We Collect

WhiteRocket does not collect personal information from users.

The app currently does not include:

- User accounts
- Advertising SDKs
- Analytics SDKs
- Crash reporting SDKs
- In-app payments
- Developer-operated VPN servers

## Information Stored On Your Device

WhiteRocket stores imported VPN/proxy server configurations locally on your device. These configurations may include server names, hostnames, ports, UUIDs, usernames, passwords, transport settings, and the original imported config text.

This information is used only to display your server list and start the VPN/proxy connection you select. It is not uploaded to WhiteRocket or to the app developer.

You can remove saved server configurations in the app. Uninstalling the app also removes its local app data from your device.

## VPN And Network Traffic

When you connect using WhiteRocket, Android routes network traffic through the VPN/proxy configuration you selected.

WhiteRocket does not operate the VPN/proxy servers used by the app. Users provide their own server configurations. The operator of the selected server may be able to process or observe network metadata, and in some cases traffic content, depending on the server setup, destination websites, and encryption used.

You are responsible for choosing a server provider or self-hosted server that you trust.

## QR Code And Image Import

WhiteRocket can import server configurations by:

- Scanning a QR code with the camera
- Reading a QR code from an image selected by the user
- Manually pasting a proxy configuration link

Camera access is used only for QR code scanning. Image access is used only for the image that you choose through Android's file picker.

## Permissions

WhiteRocket may request the following Android permissions or system access:

- VPN permission: required to create the Android VPN connection.
- Internet/network access: required for proxy/VPN connectivity.
- Camera access: used only when scanning QR codes.
- Notification access for foreground service notifications: used to show VPN connection status and allow disconnecting.

## Third-Party Components

WhiteRocket uses open-source components, including Xray/libXray for VPN/proxy functionality and ZXing for QR code scanning/decoding.

These components are used inside the app to provide VPN/proxy and QR import features. WhiteRocket does not use them to collect analytics or advertising data.

## Children's Privacy

WhiteRocket is not directed to children under 13. The app does not knowingly collect personal information from children.

## Changes To This Policy

This privacy policy may be updated as WhiteRocket changes. Updates will be published in this repository.

## Contact

For privacy questions, contact the developer through GitHub:

https://github.com/dl-maxwang

# WhiteRocket 隐私政策

生效日期：2026 年 7 月 5 日

WhiteRocket 是一款 Android VPN/代理客户端，用户可以导入并使用自己的服务器配置。

## 我们收集的信息

WhiteRocket 当前不收集用户个人信息。

当前版本不包含：

- 用户账号
- 广告 SDK
- 分析统计 SDK
- 崩溃上报 SDK
- 应用内支付
- 开发者运营的 VPN 服务器

## 本地保存的信息

WhiteRocket 会在设备本地保存用户导入的 VPN/代理服务器配置。这些配置可能包含服务器名称、域名、端口、UUID、用户名、密码、传输设置和原始配置文本。

这些信息只用于显示服务器列表和启动用户选择的 VPN/代理连接，不会上传给 WhiteRocket 或开发者。

## VPN 和网络流量

连接后，Android 会将网络流量通过用户选择的 VPN/代理配置转发。

WhiteRocket 不运营 VPN/代理服务器。服务器由用户自行提供或选择。服务器运营方可能会根据服务器配置、目标网站和加密方式看到部分网络元数据，某些情况下也可能处理流量内容。

请只使用你信任的服务器或服务提供方。

## 二维码和图片导入

WhiteRocket 支持通过摄像头扫描二维码、选择二维码图片，或手动粘贴代理链接导入配置。

摄像头权限仅用于二维码扫描。图片访问仅用于用户通过 Android 文件选择器主动选择的图片。

隐私相关问题可以通过 GitHub 联系开发者：
